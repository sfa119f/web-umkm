# Web UMKM
Project ini merupakan web aplikasi yang membantu UMKM khususnya di perkebunan untuk memasarkan produknya dengan nama ***Warung Ijo***. Project ini dibangun menggunakan *PostgreSQL*, *ExpressJS*, *VueJS*, dan *NodeJS*.
