# backend-web-umkm

## Setup Database
- Install [PostgreSQL](https://www.postgresql.org/download/)
- Make database
- Import ```database/warung_ijo.pgsql``` ke database
- Konfigurasi ```dbConfig.js``` sesuai database yang dimiliki

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
node index.js
```
